# Data-Analysis
Interactive Microsoft Excel dashboard analyzing Super Store sales performance using Pivot Tables, Pivot Charts, and Slicers.
Super Store Sales Analysis Dashboard
Overview

The Super Store Sales Analysis Dashboard is an interactive Microsoft Excel project designed to analyze sales performance and customer behavior using data visualization techniques. The dashboard provides insights into monthly sales trends, customer demographics, regional performance, order status, and sales channel contribution through interactive filters and charts.

Business Objective

The objective of this project is to analyze sales data and generate actionable business insights by answering key business questions related to customer purchasing behavior, sales performance, and operational efficiency. The dashboard enables users to interact with the data and make informed business decisions.

Key Insights
Women contributed approximately 64% of total sales, while men contributed 36%.
Adult customers represented the largest customer segment.
March recorded the highest sales and order volume.
Amazon was the leading sales channel, followed by Myntra and Flipkart.
Maharashtra generated the highest revenue among all states.
The majority of orders were successfully delivered, with only a small percentage being cancelled, returned, or refunded.
Interactive slicers allow users to analyze sales based on month, sales channel, and product category.
